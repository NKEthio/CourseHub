# **App Name**: CourseHub

## Core Features:

- Course Catalog: Display a list of available courses with relevant details, like title, short description, teacher name, and average rating.
- Lesson Videos: Allow teachers to input the video URLs (e.g., YouTube) for each lesson.
- AI Readings: AI powered "suggested readings" tool, recommending existing content (articles, videos, etc.) that complement the current course. Generative AI determines the relevancy of content to suggest. Includes the ability for a human editor to override the AI suggestions

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and knowledge, suitable for an educational platform.
- Background color: Light gray (#EEEEEE), providing a clean and neutral backdrop to ensure readability and focus on content.
- Accent color: Yellow-orange (#FFAB40), used for interactive elements like buttons and ratings to draw attention and add a touch of vibrancy.
- Clean and modern typography with a focus on readability. Use a sans-serif font for headings and body text.
- Consistent and simple icons to represent course categories, resources, and actions.
- Clear, intuitive navigation with well-defined sections for courses, lessons, and user profiles.
- Subtle animations and transitions to enhance user experience without being distracting.
